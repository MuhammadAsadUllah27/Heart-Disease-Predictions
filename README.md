# Heart-Disease-Predictions
## Task Objective 
The main task objective of this project is that can predict that the person can have the heart disease or not, based on the symptoms. And it can also help us in prediction of the disease in early stage, and also help the doctor to take decision, and also reduce error in prediction so it improves health care outcomes
## Dataset Used
Heart Disease UCI Dataset is used that is available in the kaggle.
## Model Applied
Logistic Regression which is a classification model is applied.
## Key Results and Findings
Our model is performing well so that we can predict that the patient have heart disease or not. This model helps us to predict the disease at early stage.
